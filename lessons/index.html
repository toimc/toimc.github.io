我的课程

概述

Hi, 在《慕课网》录制了如下视频课程：

3小时速成 Vue2.x 核心技术：了解vue2.x的核心技术，建立前端组件化的思想

大前端-高级前端工程师，对标P6+：慕课精品成长课，培养更高思维+技术能力的综合性高端前端人才

互联网人副业指南：全网稀缺，超越市面大多副业课，从0到1掌握副业成功密码

初衷

愿景：传播技术的种子，让分享带来价值

经验：如果一个人在学习一项新技术的时候，想象的最终他是要把这项技术输出或者说是交给别人，那么他正在使用一种叫做输出式的学习方式。

好的老师，不应该是云里雾里，而是言简意赅通俗易懂，那么这句话，我之前还不是特别的理解，但是现在我非常的认同。

过程：所有录好的视频，都会重复的去听，精心的剪辑。同时，在录制之前，我会考量所讲内容是否是最佳的一种表达方式，是否能够让没有任何前端基础的同学也能够听懂也能够去理解。所以，视频总是录了又删删了又录，我相信这些努力是有价值的，因为大家在听到看到我的视频的时候，会觉得这门技术不是那么的冷冰冰，它是可以学会和掌握的。

2021年总结

同学们，爆竹声中一岁除，新的一年即将来临。Brian来做年终汇报啦~

旧的计划

首先，让我们回顾2021年的计划：

  • 课程：设计同学们所需知识内容的直播，继续保持内容的更新，参与同学答疑；——完成

    从2021年1月1日0时0分0秒至2022年1月1日0时0分0秒，共回复了584条同学的问题，所有问题均已实现闭环（已采纳，或者已回复）。

  • 教学环节：丰富教学环节，增大学习过程的交互，让学习体验提升，效果更友好。——完成

    全年直播互动，组织开源项目、电子书更新。

  • 项目：研讨微服务、Node后台、移动端（跨端）的项目储备与内容设计；——完成

    新更新内容之后，《大前端》真正做到了全端全栈全应用场景的内容。

  • 社区：完善一期社区功能，升级完整Vue3对应的代码，研讨移动端内容；——完成

    Flutter、React对于大家伙是不是比较难，但是也能窥出一点点未来的技术趋势。uniapp这样的生态，只会在国内的大环境中进行应用。

  • 研讨开源基础的框架与代码，做开源项目——基本完成，后续全部交由我来主导的更新。

2022计划

《大前端》在2021年迎来了更新，在过去的一年中，我们更新了Flutter移动端、Electron桌面端、React进阶，同时把小程序部分的mpvue的内容进行了翻新（翻新时长20小时），更新课程电子书https://front-end.toimc.com，以上是课程总结，总体来说对于Brian，不是100分，我觉得还可以做的更好。所以，我们2022年的计划：

  • 课程更新与直播不变，形式可能会去探索短视频、短博文等形式；
  • 开源项目：中后台的思路不变，转向Vue3/TS/Vite/Flutter/Electron，迎接Vue3的正式版本，我一直想通过“开源项目”的形式，让更多的技术小伙伴能受益——这就是开源精神？
  • 推进自己的想法：传播技术的种子，让分享带来价值，让“内向”的程序员们，也能通过技术搞点“外快”~
  • 内推、面试、求职、商城、占卜相关的板块上线toimc小程序；
  • 可能会：开发toimc公众号（服务号）->为了解决消息推送的问题、toimc技术社区（App）；

一直秉持着“不忘初心，方得始终”服务同学的态度，为的就是把内容质量做好。

新的开始

前端技术行业内卷，不必过于焦虑，你可以做到居安思危，未雨绸缪，生活还是要继续。如果不清楚自己想的是什么时候，就闭上眼，休息一下。想清楚比一开始就动手做，效率要高很多，也会少出错。除了工作，大家应该有一个自己非常执着与爱好的事情，这个就是精神食粮。帮助小伙伴们寻找那个心中挚爱的事情，也会成为2022年我的主旋律。

送给大家一段，自己非常喜欢的话，它常常勉励着我。

“ 人生当中成功只是一时的，失败却是主旋律，但是如何面对失败却把人分成了不同的样子，有的人会被失败击垮，有的人能够不断的爬起来，继续向前。我想真正的成熟应该并不是追求完美，而是直面自己的缺憾，这才是生活的本质。罗曼罗兰说过的，这个世上只有一种真正的英雄主义。那就是认清生活的真相，并且仍然热爱他。 ”

如果你还未到顶峰或者已从顶峰跌落，那就继续向上攀爬，甚至从头再来吧。难道向上攀爬的那条路不是比站在顶峰更让人热血澎湃吗？相信自己还有更多的机会。

2021年再多的苦难它已经过去，去睡个好觉吧，今晚之后，是2022年，活在当下~鸡血的一年，马上开始。

最后，值此新春佳节，Brian祝您及家人虎年大吉[福]，虎虎生威[爆竹]，幸福安康，万事如意！

2020年总结

告别了2020磨难的一年，新的一年里大家有没有准备好继续出发了呢？

《大前端》课程已陪伴了大家一年，在过去的一年中，我们团队秉持着“不忘初心，方得始终”服务同学的态度，为大家更新了视频的内容，并扩展了Vue3的内容，具体数据如下：

  1. 先后对第一批推出的100+小时视频，其中152节视频进行了精剪，加速，内容修正；
  2. 加入扩展的Vue3的内容12+小时，并预计后续继续优化，扩展部分TypeScript知识；
  3. 从2020年1月1日0时0分0秒至2021年1月1日0时0分0秒，共回复了1252条同学的问题，所有问题均已实现闭环（已采纳，或者已回复）；
  4. 统计问答区的高频问题144条，并更新对应的视频的内容，或加入字幕；梳理对出对学习有帮助的问答，课程中让大家更方便查看。
  5. 补充Q群中的资料，多次发布重要的技术更新公告通知，并配合慕课完善了资料的访问方式；
  6. 2020年完成了6次直播，收获了同学们的喜爱以外，帮助同学们修改简历，加强沟通，调整“悲观”的心态，已有多位同学成为我们的一员；

2021年的计划

  1. 课程：设计同学们所需知识内容的直播，继续保持内容的更新，参与同学答疑；
  2. 教学环节：丰富教学环节，增大学习过程的交互，让学习体验提升，效果更友好。
  3. 项目：研讨微服务、Node后台、移动端（跨端）的项目储备与内容设计；
  4. 社区：完善一期社区功能，升级完整Vue3对应的代码，研讨移动端内容（预计）；研讨开源基础的框架与代码，做开源项目；
